# 📋 Шаблон карточки продукту EMET — для медвідділу

**Версія:** 1.0 (24.04.2026)
**Hто заповнює:** методист медвідділу + Sales Director (sales playbook розділ)
**Цільовий час на заповнення:** 30-40 хв на продукт
**Куди потрапляє:** RAG бази бота → відповіді менеджерам у Telegram-боті

---

## 🎯 Принципи

1. **Коротко і конкретно** — 1 факт = 1 рядок. Без маркетингової води («революційна формула», «преміум-якість»).
2. **Числа цифрами** — «18 міс», не «вісімнадцять місяців». «20 мг/мл», не «висока концентрація».
3. **Якщо не знаєш — пропусти** — краще пусте поле ніж вигадане.
4. **Розділ A** (об'єктивні факти) — обов'язковий. **Розділ B** (sales playbook) — бажаний.
5. **ESSE — космецевтика, не інʼєкція** — НЕ вживати слово «CE». Замість — INCI, дерматологічні тести.

## 💼 РОЗДІЛ B — Sales Playbook (Sales Director + методист)

### B.1. Унікальна торгова перевага (USP)

| Поле | Формат | Приклад |
|---|---|---|
| **USP в 1 реченні** | <30 слів | Ellansé — єдиний у портфелі біостимулятор з КЛІНІЧНО передбачуваною тривалістю до 24 міс завдяки PCL gold standard |
| **VS Sculptra** | 2-3 факти | Ellansé: миттєвий об'єм + поступова стимуляція. Sculptra: тільки стимуляція, без об'єму. Менше процедур у курсі (1 vs 3-4). |
| **VS AestheFill** | 2-3 факти | PCL vs PDLLA: PCL має кращу прогнозованість, повна біодеградація. |
| **VS Petaran** | 2-3 факти | Ellansé — для волюмізації, Petaran — для якості шкіри. Це РІЗНІ механізми, не конкуренти. |

### B.2. Топ-5 заперечень + готові аргументи

| Заперечення | Killer answer (1-2 речення) |
|---|---|
| **«Дорого / в 3 рази дорожче за Petaran»** | 1 преміум-процедура замість 3-4 базових = вищий чек, менше слотів, задоволений пацієнт повертається |
| **«Не чула про бренд / нова марка»** | Ellansé входить у портфель Sinclair Pharma (UK) — gold standard PCL у Європі з 2009 року |
| **«У мене вже є Sculptra»** | Sculptra — тільки стимуляція без об'єму, потрібно 3-4 процедури. Ellansé — 1 процедура з миттєвим ефектом + стимуляція |
| **«Клієнти бояться довго діючих філерів»** | Повна біодеградація (CO₂+H₂O), доказана клінічними дослідженнями. На відміну від PMMA — слідів НЕМАЄ |
| **«Складно прогнозувати результат»** | Передбачуваність — головна перевага PCL. 18 vs 24 міс — точні цифри, не «приблизно рік» як у конкурентів |

### B.3. Killer phrases (3-5 для лікаря в кабінеті)

1. *«Ellansé — це не просто філер, це інструмент структурного омолодження на 1.5-2 роки»*
2. *«Один продукт замість трьох — економія слотів вашого графіку та задоволеність пацієнта»*
3. *«Це той випадок коли пацієнт повертається до вас за результатом, а не за корекцією»*

### B.4. Situational hooks (тригери для допродажу)

| Якщо лікар згадав... | Запропонуй... |
|---|---|
| «Зменшення об'ємів обличчя 40+» | Ellansé S для скул + Petaran для якості шкіри (комбо) |
| «Хочу довгограючий ефект» | Ellansé M (24 міс) замість стандартного філера |
| «Працюю з преміум-сегментом» | Ellansé + домашній догляд ESSE Plus (повний цикл) |
| «Пацієнтка не хоче часто колоти» | Ellansé M — 1 процедура на 2 роки |

### B.5. Cross-sell комбінації

- **Ellansé + Petaran** — структура + якість шкіри (різні зони!)
- **Ellansé + Vitaran i** — об'єм + поверхневе омолодження
- **Ellansé + Neuramis** — об'єм + філер у губи / періорбіту
- **Ellansé + ESSE домашній догляд** — підтримка результату між процедурами

### B.6. Маржинальність для клініки

| Параметр | Значення |
|---|---|
| **Закупкова ціна для клініки** | $X (питати у комерційного відділу) |
| **Рекомендована вартість процедури** | $Y |
| **Маржа клініки** | ~Z% |
| **LTV пацієнта (середній)** | 1 процедура / 18-24 міс + супутні |

### B.7. Профіль ідеального пацієнта

- Вік: **40-55 років**
- Проблема: втрата об'ємів середньої третини обличчя
- Платоспроможність: преміум-сегмент
- Очікування: довгограючий результат, мінімум процедур
- НЕ підходить: молода шкіра <35, аутоімунні, очікування миттєвого «філерного» ефекту

### B.8. ⛔ ЗАБОРОНЕНІ фрази (юридично/етично)

- **НЕ казати** «вічний філер» — продукт біодеградує
- **НЕ казати** «безпечно для всіх» — є протипоказання
- **НЕ казати** «гарантований результат» — індивідуальна реакція
- **НЕ обіцяти** «без побічних ефектів» — реабілітація 3-7 днів
- **НЕ використовувати** «mass-market» / «аптечний аналог» (для будь-якого EMET-продукту)

**Repeating header у chunks** (вирішує scope=line vs product плутанину):

# 📝 Що важливо для медвідділу знати

1. **Заповнення розділу A** — це **обов'язково**, без нього бот не зможе давати точні відповіді
2. **Заповнення розділу B** — **бажано**, але якщо немає часу — Sales Director може заповнити сам
3. **Якщо нема даних** — пропустіть поле, не вигадуйте. Бот скаже «уточніть у медвідділі» — це краще ніж галюцинація.
4. **30-40 хв на продукт** — це реалістично якщо мати інструкцію виробника під рукою
5. **Оновлення раз на квартал** або при зміні інструкції виробника
