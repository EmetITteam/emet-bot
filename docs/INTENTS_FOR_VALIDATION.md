## Що потрібно від вас

EMET-бот дає ~10% галюцинацій та неправильних структур. Архітектурний рефакторинг запланований на 15-17.04. Для нього потрібні **ваші експертні структури відповідей** на типові запити менеджерів.

Для кожного intent нижче:
1. **Подивіться мою draft-структуру**
2. **Якщо згодні** — поставте ✅ в колонці "Статус"
3. **Якщо треба змінити** — напишіть у "Коментар експерта" що саме:
   - які пункти прибрати / додати
   - які формулювання змінити
   - що забороняється писати взагалі (legal/compliance)

```

Висновок: це [нормальна / ненормальна] реакція, свідчить про [активну дію / проблему]

✅ Це нормально чи турбуватись?
```

### 2. `clinical_no_result` — "Не бачу результату"
**Хто валідує:** 👨‍⚕️ Med Director

**Приклад з логів (14.04):**
> _"Petaran - не бачу результату, інші препарати дають більш виражений ліфтинг"_

**Що робив бот (неправильно):**
Пропонував перейти на інший продукт (VITARAN) з критикою Petaran. Не перевіряв чи правильно застосовано протокол.

**Моя draft-структура:**
```
🔍 Перевірка правильності застосування:
1. Протокол: скільки процедур проведено? Який інтервал? ([VERBATIM з документа])
2. Техніка введення: [правильна для продукту]
3. Зона введення: [показання за протоколом]
4. Підготовка пацієнта: [вимоги]

⏱️ Реалістичні очікування за термінами:
• N тижнів: [перший ефект]
• N місяців: [основний ефект]
• Повний результат: [термін]
(VERBATIM з документа препарату)

💪 Як підсилити ефект:

• Комбо з [препарат Y] — для [ефект]

🎯 Як подати лікарю:

давайте перевіримо протокол, можливо треба [корекція].»

⛔ НЕ:
• Не переключай на інший препарат без аналізу
• Не критикуй Petaran / наш продукт
• Не обіцяй гарантій результату
```

### 4. `combo_with_product` — Комбо-протоколи
**Хто валідує:** 👨‍⚕️ Med Director

**Приклад з логів (14.04):**
> _"комбо з Petaran"_

**Що робив бот (неправильно):**
Перефразовував "не можна в одну зону" → "не можна комбінувати взагалі". Писав `[інформація відсутня]` як плейсхолдер. Іноді вигадував протоколи з іншими препаратами.

**Моя draft-структура:**
```
🔗 Комбо-протоколи: [Препарат X]

1. [Препарат X] + [Препарат Y]
   • Схема: [VERBATIM з документа — з інтервалом, зонами, умовами]
   • Показання: [з документа]
   • ⚠️ Обмеження: [VERBATIM, БЕЗ перефразування]

2. [Препарат X] + [Препарат Z]
   • Схема: [VERBATIM]
   • ...

[і так далі для ВСІХ комбінацій з документа]

⛔ Правила цитування:
• Тільки точні слова з документа
• "не в одну зону" ≠ "не можна"
• Якщо немає даних для пункту — ПРОПУСТИ, не пиши "[інформація відсутня]"
```

**Статус:** ⬜

**Коментар Med Director:**
> _[Чи правильний підхід verbatim? Які ще комбінації критичні?]_

### 5. `objection_grey_market` — "Сірий ринок дешевше"
**Хто валідує:** 💼 Sales Director + ⚖️ Legal

**Приклад з логів (08.04):**
> _"косметолог не хоче працювати з Neuramis, бо його багато на сірому ринку за більш низькою ціною"_

**Що робив бот (частково правильно):** Дав fear-based аргументи про FDA, якість. Але не згадував юридичні ризики клініки.

**Моя draft-структура:**
```
⚡ SOS: сірий ринок препарату

💬 Крок 1 — відкриваюче питання:
«Чи знаєте скільки коштує одне ускладнення від несертифікованого препарату
для клініки — юридично і репутаційно?»

🏥 Fear-based (юридичні ризики):
1. Клініка не може довести походження препарату при перевірці МОЗ
2. У разі ускладнення — повна відповідальність клініки (страховка не покриває сірий ринок)
3. Пацієнт має право на позов про відшкодування шкоди + моральна шкода

💰 Фінансова математика:
• Економія на сірому ринку: Xгрн / шприц
• Ризик одного позову: 50-500К грн + репутаційні збитки

🔑 Killer phrase:
«Економія 500 грн = ризик 500К грн + закрита клініка на 3 місяці»

📲 Наступний крок:
• Привезти сертифікати + документи про офіційне постачання
• Показати кейс юриста клініки / МОЗ-перевірки

⛔ НЕ:
• Не обвинувачуй лікаря в нелегальній діяльності
• Не лякай реальними назвами клінік-прикладів
```

**Статус:** ⬜

**Коментар Sales + Legal:**
> _[Чи правильні юридичні формулювання? Які ризики ще додати? Які фрази точно заборонені з legal point of view?]_
